# Python Train Workspace

This workspace contains a collection of small AI and Python projects:

- `ai-chatbot/` - a notebook-based chatbot example
- `ai-story-generator/` - a story generation app with backend and frontend pieces
- `finance_tech_finetune/` - fine-tuning scripts, data, and notebooks for finance and tech domains
- `lib-manage/` - a library management API project with Docker support
- `proj-1-aimodel-serving/` - an AI model serving project with backend and frontend
- `sentiment-analyser/` - a sentiment analysis app

## Contents

- Notebook experiments and datasets
- Backend Python applications
- Frontend web apps
- Training, evaluation, and inference scripts

## Getting Started

Open the subproject you want to work on and follow the instructions in its own README or source files. If a project does not include setup notes yet, start with its `requirements.txt` or application entry point.
